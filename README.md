# Tkinter-Practice_Glasses-or-No-Glasses

## 개요

- Tkinter, Keras 활용 이미지 인식 프로그램
- Keras를 활용해 안경 쓴 사람과 안 쓴 사람을 학습시키고 학습시킨 모델을 적용해 이미지 속 사람이 안경을 썼는지 쓰지 않았는지에 대한 판단 결과를 보여줌

## 파일 목록

### test_noglasses.jpg` - 안경 쓰지 않은 사람 테스트 이미지
